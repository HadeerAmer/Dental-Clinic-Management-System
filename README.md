# Dental Clinic Management System

A comprehensive dental clinic management system developed in C# as an Object-Oriented Programming (OOP) course project. This desktop application streamlines clinic operations by managing patients, appointments, treatments, and administrative tasks with persistent database storage.

## Features
- **Patient Management**: Register and maintain patient records with medical history
- **Appointment Scheduling**: Book, modify, and track patient appointments
- **Treatment Tracking**: Record and monitor dental treatments and procedures
- **Doctor Management**: Manage dentist profiles and schedules
- **Billing System**: Generate invoices and track payments
- **Database Integration**: Persistent data storage and retrieval
- **Reports**: Generate various reports for clinic operations
